Changelog v1.0

It is update time here on Limitless Roleplay!

Today's update brings the long awaited factions system update! And boy is there a lot here, so sit back and get ready. This system is a collection of many hours of blood, sweat, tears, and cursing.

The new faction system is a completely new framework built from scratch for Limitless. No longer will you have to choose between a public service job, and a legal paying job. From this point forward, you'll be able to do both as you please!

A WHOLE NEW FLEET

President Josh has came through with a ton of funding for each department! All departments have been granted an entirely new fleet of police vehicles.

DEPARTMENT REMODELING PROJECTS

Funding has also been used to completely remodel all public service departments in the city. Some hospitals have been given a remodel as well, with more to come.

Alright, I'm tired of all this fancy ass shit. Heres everything else you're getting in this update.

  • Tear gas launcher for leos, replacing the grenade launcher.
  • New Fire and Medical department HQ
  • Fire system to actually add some fire work for SAFMD to do.
  • Ability to select peds through the plastic surgery menu. To do this, you'll want to arrow through the "genders" category.
  • EUP has been OVERHAULED. The command is no longer required, and it can be accessed through your PD MENU.
  • The Impound lot in Sandy has been revamped and relocated. You'll no longer be able to spawn cars on top of other vehicles, and will need to move anything blocking the spawner.
  • FiveM has supposedly made significant bug fixes to AI on Onesync. I've enabled it on my test server, and so far no issues there. Could change with a large player base though. AI will be enabled for testing on this update, and if it poses an issue will be disabled again.

    There is probably more that i'm forgetting, because i'm exhausted from this update.

Big props to Lance Good for his hard work on this update.
